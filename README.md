# idioma


## Funcionalidad

Obtiene el porcentaje de idiomas que contiene una base.

## Como se Utiliza

Para utilizar el programa es muy sencillo, primero se descarga él .ZIP del repositorio, luego se extraer la carpeta y archivos del .ZIP.

Dentro de la carpeta encontraras el programa idioma.R, este programa es ejecutado a través de un doble click en idioma.bat
